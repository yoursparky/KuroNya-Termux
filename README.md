---

A stylish terminal-based AI built with Node.js + Groq API.

Created by Sparky.

---

✨ Features

- 🧠 Groq LLM Integration (llama-3.1-3b-instant)
- 🎭 Animated ASCII Avatar (emotion-based)
- 📊 Live Emotion Progress Bar
- 🌧️ Matrix Rain Mode ("/matrix")
- 🖼️ PNG Avatar Generator ("/export")
- 🔊 Termux TTS Voice
- 💾 Persistent Memory System (JSON based)

---

🛠 Tech Stack

- Node.js
- Groq API
- Axios
- Chalk
- Canvas
- Termux TTS

---

🚀 Commands

Command| Description
/matrix| Activate Matrix rain effect
/export| Save current avatar as PNG
/mood <type>| Change AI personality mood

---

⚙️ Setup

1. Clone repo
2. Run "npm install"
3. Add your Groq API key inside "index.js"
4. Start with:

node index.js

---

🔧 Termux Setup

1. pkg install && pkg upgrade
2. pkg install nodejs
3. mkdir ai
4. cd ai
5. nano index.js (Paste code Press CTRL X→ Y → Enter)
6. nano package.json (Same Step)
7. npm install
8. npm start
